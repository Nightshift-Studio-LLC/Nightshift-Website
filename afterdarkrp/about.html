About The Project

About

AfterDarkRP is Nightshift's sandbox roleplay server stack. This site content reflects the current project in `D:\Nightshift\ServerHosting\darkrp_ns`.

Project Snapshot

Engine + Stack

  • s&box project (`darkrp_ns.sbproj`)
  • C# gameplay systems
  • Razor-based in-game UI panels
  • Scene/prefab-driven world entities

Major Gameplay Modules

  • Wallet, payday, bank persistence
  • Jobs catalog + role switching
  • Taze/stun/detain/prison justice flow
  • Crime record + fines + terminal UI
  • Death cash drop + pickup bag entity

UI Surfaces

  • HUD includes health ring, taze ring, cash, and current job.
  • Player Menu tabs: Wallet, Jobs, Entities, Players, Settings.
  • Player Pockets panel currently exposes 20 inventory slots.
  • ATM and Crime Terminal run as dedicated focused panels.

Near-Term Roadmap

  • Finalize job loadout assignment with custom tools (PhysGun/ToolGun scaffolds).
  • Expand police/criminal balancing around taze values and sentence pacing.
  • Promote placeholder entity tab items to fully networked gameplay entities.
  • Ship additional map interactions around courthouse/prison spawn markers.
