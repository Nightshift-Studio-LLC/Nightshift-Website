Unreal Editor Utility

Landsnap

Tired of manually nudging actors to sit correctly on terrain or surfaces?

LandSnap is a lightweight Unreal Editor utility that instantly snaps and aligns selected actors and splines to Landscapes or designer-defined tagged surfaces, saving time and eliminating tedious manual adjustments during level design.

No complex setup. No bloated systems. Just fast, predictable placement.

ðŸ”§ What LandSnap Does

Snap Targets

  • Snaps selected Actors, Spline Components (paths, roads, guides), and Spline Mesh Components
  • Snap to Landscape surfaces
  • Snap to tagged actors or components (author-controlled surface gating)
  • Supports stacking / placement on valid tagged surfaces

Placement & Alignment

  • Pivot-aware bottom alignment
  • Optional alignment to surface normals
  • Optional spline up-vector / direction alignment
  • Handles start and end spline points
  • Z Offset support
  • Configurable trace distances
  • Supports multiple actors at once
  • Supports spawned actors below Landscape

Editor Integration

  • Toolbar buttons
  • Editor Preferences settings panel
  • Hotkey support (command-based)
  • Defaults: CTRL + Down, SHIFT + Down
  • Undo / Redo via scoped transactions
  • Editor-only (no runtime impact)
  • World Partition & HLOD safe
  • Optional debug line draw + logging (Log LogLandSnap Verbose)
